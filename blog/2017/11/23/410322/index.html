    тЈЃтіатИѓтю░жЄЇтіЃтѕєтЏътюЪтю░жЮбуЕЇУѕЄтЈЃтіатЇђТ«хтЙхТћХтѕєтЏътюЪтю░жЮбуЕЇС╣ІТ»ћУ╝Ѓ

    тЁДТќЄ¶

    тюЪтю░ТЅђТюЅТгіС║║тЈЃтіатИѓтю░жЄЇтіЃтѕєтЏътюЪтю░жЮбуЕЇС╣ІУеѕу«ЌТќ╣Т│ЋУЕ│тдѓтИѓтю░жЄЇтіЃт»дТќйУЙдТ│Ћугг29ТбЮжЎёС╗ХсђѓтюЪтю░ТЅђТюЅТгіС║║тЈЃтіатЇђТ«хтЙхТћХтѕєтЏътюЪтю░жЮбуЕЇС╣ІУеѕу«ЌТќ╣Т│ЋУЕ│тдѓтюЪтю░тЙхТћХТбЮСЙІТќйУАїу┤░тЅЄугг50ТбЮжЎёС╗ХсђѓС║їУђЁС╣ІтЁгт╝ЈжаЌуѓ║УцЄжЏю№╝їтИИтЏ░ТЊЙтГИућЪ№╝їтЏаТГцуГєУђЁт░ЄтЁХтЁгт╝Ју░Атїќ№╝їС╗ЦтѕЕтГИу┐њсђѓ

    Рђб (СИђ) тИѓтю░жЄЇтіЃ№╝џтЈЃтіатИѓтю░жЄЇтіЃ№╝їтјЪТЅђТюЅТгіС║║ТЄЅУ▓аТЊћС╣ІжаЁуЏ«№╝їтїЁТІгУЄеУАЌтю░уЅ╣тѕЦУ▓аТЊћсђЂСИђУѕгУ▓аТЊћтЈіУ▓╗ућеУ▓аТЊћуГЅСИЅжаЁсђѓт»дТќйтИѓтю░жЄЇтіЃТЎѓ№╝їтјЪТЅђТюЅТгіС║║ТЄЅтѕєжЁЇжЮбуЕЇС╣ІУеѕу«ЌТќ╣Т│Ћу░АтїќтдѓСИІ№╝џ

    1. жЄЇтіЃтЅЇтЙїт«Ќтю░тю░тЃ╣СИіТ╝▓ујЄ(A)=[тюќуЅЄ1]

    2. жЄЇтіЃтЇђСИђУѕгУ▓аТЊћС┐ѓТЋИ(B)=[тюќуЅЄ2]

    3. жЄЇтіЃтЇђУ▓╗ућеУ▓аТЊћС┐ѓТЋИ(C)=[тюќуЅЄ3]

    4. тљёт«ЌтюЪтю░жЄЇтіЃтЙїТЄЅтѕєжЁЇС╣ІжЮбуЕЇ(D)=[жЄЇтіЃтЅЇтјЪТюЅС╣Іт«Ќтю░жЮбуЕЇ├Ќ[тюќуЅЄ4]№╝ЇУЄеУАЌтю░уЅ╣тѕЦУ▓аТЊћ]├Ќ[тюќуЅЄ5]

    Рђб (С║ї) тЇђТ«хтЙхТћХ№╝џтюЪтю░тЙхТћХТбЮСЙІугг44ТбЮугг1жаЁугг1ТгЙУдЈт«џ№╝їУбФтЙхТћХтюЪтю░ТЅђТюЅТгіС║║жаўтЏъТіхтЃ╣тю░С╣ІжЮбуЕЇ№╝їућ▒УЕ▓у«АуЏ┤УйётИѓТѕќуИБ№╝ѕтИѓ№╝ЅСИ╗у«АТЕЪжЌюТїЅтЁХТЄЅжаўтю░тЃ╣УБютёЪУ▓╗УѕЄтЇђТ«хтЙхТћХУБютёЪтю░тЃ╣уИйжАЇС╣ІТ»ћујЄУеѕу«ЌтЁХТЄЅжаўС╣ІТгітѕЕтЃ╣тђ╝№╝їСИдС╗Цт»джџЏжаўтЏъТіхтЃ╣тю░С╣Ітќ«СйЇтю░тЃ╣Тіўу«ЌС╣ІсђѓТ║ќТГц№╝їт»дТќйтЇђТ«хтЙхТћХТЎѓ№╝їтЂЄт«џтЇђТ«хтЙхТћХу»ётюЇтЁДуёАУЙ▓тю░жЄЇтіЃтЇђ№╝їтјЪТЅђТюЅТгіС║║жаўтЏъТіхтЃ╣тю░жЮбуЕЇС╣ІУеѕу«ЌТќ╣Т│Ћу░АтїќтдѓСИІ№╝џ

    1. тЁетЇђжаљУеѕТіхтЃ╣тю░жЮбуЕЇ=тЁетЇђС╣ІтЙхТћХтюЪтю░уИйжЮбуЕЇ├ЌТіхтЃ╣тю░Т»ћСЙІ№╝ѕ40%~50%№╝Ѕ

    2. жаљУеѕТіхтЃ╣тю░уИйтю░тЃ╣=╬БУдЈтіЃСЙЏТіхтЃ╣тю░тѕєжЁЇС╣ІтљётѕєжЁЇУАЌт╗ЊжЮбуЕЇ├ЌтљёУЕ▓тѕєжЁЇУАЌт╗ЊУЕЋт«џС╣Ітќ«СйЇтю░тЃ╣

    3. тЇђтЁДтљётјЪтюЪтю░ТЅђТюЅТгіС║║жаўтЏъТіхтЃ╣тю░С╣ІТгітѕЕтЃ╣тђ╝=[тюќуЅЄ6]

    4. тљётјЪтюЪтю░ТЅђТюЅТгіС║║жаўтЏъТіхтЃ╣тю░С╣ІжЮбуЕЇ=[тюќуЅЄ7]

    Рђб (СИЅ) Т»ћУ╝Ѓ№╝џ

    1. тИѓтю░жЄЇтіЃтѕєжЁЇтюЪтю░С╣ІТдѓт┐х№╝їС┐ѓтЁѕт░▒жЄЇтіЃтЁетЇђжЮбуЕЇТЅБжЎцтЁгтЁ▒УеГТќйућетю░У▓аТЊћУѕЄУ▓╗ућеУ▓аТЊћтЙї№╝їтєЇТїЅтю░тЃ╣Т»ћСЙІтѕєжЁЇухдтљётјЪТЅђТюЅТгіС║║№╝ЏтЏаТГц№╝їтИѓтю░жЄЇтіЃт▒гТќ╝Тћ┐т║юУѕЄтю░СИ╗тЁ▒тљїжќІуЎ╝тюЪтю░№╝їтю░СИ╗тѕєТЊћтЁгтЁ▒УеГТќйућетю░тЈіжЄЇтіЃУ▓╗уће№╝їСИдтѕєС║ФжќІуЎ╝тѕЕуЏісђѓтЇђТ«хтЙхТћХтѕєжЁЇтюЪтю░С╣ІТдѓт┐х№╝їС┐ѓтЁѕт░▒тЙхТћХтЁетЇђтюЪтю░С╣ўС╗ЦТіхтЃ╣тю░Т»ћСЙІ№╝їтєЇТїЅтЙхТћХУБютёЪтю░тЃ╣Т»ћСЙІтѕєжЁЇухдтљётјЪТЅђТюЅТгіС║║№╝ЏтЏаТГц№╝їтЇђТ«хтЙхТћХСИђТќ╣жЮбтю░СИ╗тц▒тј╗тюЪтю░№╝їтЈдСИђТќ╣жЮбтю░СИ╗тѕєС║ФтюЪтю░жќІуЎ╝УѕЄтю░уЏ«У«іТЏ┤С╣ІтѕЕуЏісђѓ

    2. тИѓтю░жЄЇтіЃтѕєтЏътюЪтю░С╣ІУеѕу«ЌтЁгт╝ЈУ╝ЃтљѕуљєсђѓтЇђТ«хтЙхТћХтѕєтЏътюЪтю░С╣ІУеѕу«ЌтЁгт╝ЈСИГ№╝їТюЅжЌюТіхтЃ╣тю░Т»ћСЙІ(уЈЙУАїУдЈт«џ40%~50%)Тгау╝║уљєУФќТа╣ТЊџ№╝їтц▒С╣ІТГдТќисђѓ

    3. тИѓтю░жЄЇтіЃтѕєтЏътюЪтю░Тюђт░Љ55%№╝їтЇђТ«хтЙхТћХтѕєтЏътюЪтю░Тюђт░Љ40%№╝їТюђтцџ50%сђѓТЋЁтИѓтю░жЄЇтіЃтѕєтЏътюЪтю░У╝Ѓтцџ№╝їтЇђТ«хтЙхТћХтѕєтЏътюЪтю░У╝Ѓт░Љсђѓ

    4. т»дТќйтИѓтю░жЄЇтіЃ№╝їтю░СИ╗тѕєтЏътюЪтю░тЈфС║ФТюЅтюЪтю░жќІуЎ╝С╣ІТ╝▓тЃ╣тѕЕуЏі№╝їТюфС║ФТюЅтю░уЏ«У«іТЏ┤С╣ІТ╝▓тЃ╣тѕЕуЏісђѓт»дТќйтЇђТ«хтЙхТћХ№╝їтю░СИ╗тѕєтЏътюЪтю░С║ФТюЅтюЪтю░жќІуЎ╝УѕЄтю░уЏ«У«іТЏ┤С╣ІжЏЎжЄЇТ╝▓тЃ╣тѕЕуЏісђѓ

    5. тИѓтю░жЄЇтіЃтѕєтЏътюЪтю░У╝Ѓтцџ№╝їСйєжќІуЎ╝тЙїтюЪтю░Т╝▓тЃ╣У╝Ѓт░Љ№╝їТ╝▓тЃ╣СИ╗УдЂСЙєУЄфТќ╝тюЪтю░Тћ╣УЅ»№╝їТЋЁУ╝ЃуёАТџ┤тѕЕтГўтюесђѓтЇђТ«хтЙхТћХтѕєтЏътюЪтю░У╝Ѓт░Љ№╝їСйєжќІуЎ╝тЙїтюЪтю░Т╝▓тЃ╣У╝Ѓтцџ№╝їтИИТюЅТџ┤тѕЕтГўтюе№╝ЏТџ┤тѕЕСИ╗УдЂСЙєУЄфТќ╝тю░уЏ«У«іТЏ┤№╝їтЏаУђїТ╗ІућЪт«ўтЋєтІЙухљсђЂуѓњСйютюЪтю░сђѓ
