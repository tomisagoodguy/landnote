說明
(觀念剖析)

職業衛生的主管機關是行政院勞委會。 世紀之毒指以戴奧辛 (TCDD) 毒性最強見稱。

台灣常見的職業病如下所述：

(環境污染)

世紀之毒

戴奧辛

其中二、三、七、八 — 四氯戴奧辛 (TCDD) 毒性最強 ，俗稱世紀之毒。 金屬冶煉業 易排出的戴奧辛而造成環境污染，可造成皮膚、眼睛及呼吸道的刺激、頭痛、頭暈、噁心等症狀。慢性暴露會造成痤瘡、肝腫大、神經肌肉損傷、癌症

水俁病
( Minamata disease )

汞

性情變化，暴躁易怒 、 腕垂症 、 顫抖

烏腳病

砷

造成末梢血管疾病：四肢冷感、間歇性跛行、靜止疼痛、缺血性神經症、潰瘍及壞疽

痛痛病
( Itai-Itai disease )

鎘

全身骨骼疼痛、凡可尼症候群 ( Fanconi’s syndrome) ，因 近端腎小管被破壞，造成 糖尿、氨基酸尿、鈣尿， 最後腎臟萎縮、尿毒症、鈣質大量流失而骨折

威爾遜症

銅

造成肝臟與腦損傷，出現鳥翼拍擊的顫動、 四肢運動協調變差、說話不清，吞嚥困難、舞蹈症、精神異常、認知功能退化

空氣污染以 PSI 為指標， PSI>100 對身體有害。空氣污染物包括懸浮微粒、 SO2 、 NO2 、 CO 、 O3 。 SO2 造成倫敦型煙霧污染，刺激呼吸器官； PAN 造成洛杉磯型煙霧污染，刺激眼睛


溫室效應

最主要的是 CO2 ，其次是 N2O 、 CH4 和 CFCs 及 O3

這些污染物主要是燃燒石化原如煤、石油所產生的

臭氧層破壞

氟氯碳化物 ( CFCs )( 全名 Chlorofluorocarbon)

90% 的 O3 集中在平流層 。 據研究，臭氧濃度降低 l% ，皮膚癌增加 4% ，白內障發生率則增加 0.6%

酸性沉降物

硫氧化物 (SO2) 、氮氧化物 ( NO 、 NO2 ) 。雨水 PH<5.6 時稱酸雨

SO2 來自 化石燃料、火力發電廠等燃燒含硫有機物有關； NO 、 NO2 主要來自工廠高溫燃燒過程、交通工具排放廢氣有關

說明
(問題來了)以下試題摘自考選部醫師專技考試題目
<96-1-94> 職業疾病之診斷中，時常需要有流行病學資料的支持，在以下之流行病學資料之證據中，何者對於職業病診斷的必要性較低？
  1. 必須有在暴露族群之中，某疾病比對照族群為高的事實
  2. 該項工作暴露應已被證實會引起該疾病
  3. 該疾病之發作或明顯惡化是在進入該工作場所之後才發生
  4. 暴露者大多數會發作類似的疾病
解答：D
<96-1-95> 成年人慢性鉛中毒，在神經系統最容易引起下列何種症狀或病變？
  1. 失智症（ Dementia ）
  2. 小腦萎縮導致運動失調（ Cerebellar ataxia ）
  3. 巴金森氏症（ Parkinsonism ）
  4. 周邊神經病變（ Peripheral neuropathy ）
解答：D
<96-2-91> 隔離措施對下列何種傳染病的防治效果最差？
  1. 天花（ small pox ）
  2. 鼠疫（ plague ）
  3. 霍亂（ cholera ）
  4. 流行性感冒（ influenza ）
解答：B or D
<96-2-94> 以下何者不屬於水銀（汞）中毒最常見的症狀？
  1. 性情變化，暴躁易怒
  2. 步履不穩
  3. 腕垂症
  4. 顫抖
解答：B
<97-1-95> 職業性皮膚病中以何者最為常見？
  1. 色素沈積性疾病
  2. 接觸性皮膚炎
  3. 感染性皮膚疾病
  4. 結締組織疾病
解答：B
<97-2-94> 職業引起的帕金森氏症（ Parkinsonism ）與以下何種物質之暴露最有關？
  1. 汞（ Mercury ）
  2. 錳（ Manganese ）
  3. 四氯化碳（ Carbon tetrachloride ）
  4. 有機磷劑（ Organophosphate ）
解答：D
<97-2-96> 一位在噴砂工作超過 20 年的男性，若工作時未作適當的防護，他對以下那一項疾病，比一般人有較高的罹患率？
  1. 職業氣喘症 
  2. 肺結核 
  3. 肺炎 
  4. 膀胱癌
解答：B
<98-1-93> 以下何種職業暴露最常被報告造成股骨頭之壞死現象？
  1. 聯苯胺作業
  2. 全身性振動
  3. 低溫暴露
  4. 異常氣壓
解答：D
<98-1-94> 舊船解體勞工，因疲勞、全身無力、腹痛就醫，門診檢查時出現腕垂症（ wrist drop ）等手足伸肌（ extensor muscle ）無力的病變，他最可能罹患了何種金屬的中毒？
  1. 鉛
  2. 鎘
  3. 汞
  4. 錳
解答：A
<98-1-96> 一位病人因一氧化碳血紅素濃度升高而造成注意力減退。下列何種情況可造成此現象？
  1. 用含有氯化甲烷（ methylene chloride ）的溶劑清洗油漬
  2. 利用 1,1,1- 三氯乙烷（ 1,1,1-trichloroethane, methyl chloroform ）來清潔鋁製品
  3. 在電子零件工廠中經皮膚慢性暴露於乙二醇甲醚（ ethylene glycol methyl ether ）
  4. 吸入含有 carbonyl chloride 或 carbonyl bromide 的農藥
解答：A
<99-1-92> 胸部 X 光顯示肺門淋巴結腫大，且有許多小的圓形陰影散布肺野，此特徵與下列何種職業性肺病相吻合？
  1. 矽肺症
  2. 石棉塵肺症
  3. 過敏性肺泡炎
  4. 急性毒物吸入症
解答：A
<99-1-93> 一位半導體業的勞工，負責管理各種罐裝氣體，突然發生臉色蒼白、腹痛、尿液呈深褐色而就醫，理學檢查發現血壓 100/60 mmHg ，呼吸帶有大蒜味，抽血發現血液經離心之後，血清部分仍呈紅色。該患者最可能為以下何種物質之暴露所造成的中毒？
  1. 四氯化碳（ CCl4 ）
  2. 氰化鈉（ sodium cyanide ）
  3. 砷化氫（ AsH3 ）
  4. 氟化氫（ HF ）
解答：C
<99-1-94> 以下何種職業性物質暴露，曾被報告會造成鼻中隔穿孔？
  1. 硫酸氣
  2. 鉻酸電鍍液
  3. 二硫化碳
  4. 汞蒸氣
解答：A or B
<99-1-97> 日本九州曾發生水俁病（ Minamata disease ）事件，主要是由那種金屬所引起？
  1. 鎘
  2. 汞
  3. 鉛
  4. 砷
解答：B
<99-1-99> 台灣嘉南地區所發生之烏腳病，目前懷疑與何種物質有關聯？
  1. 鉛
  2. 鎘
  3. 砷
  4. 鎳
解答：C
<100-1-90> 具有接觸性皮膚炎、雷諾氏症、皮膚潰瘍病史之職業性皮膚炎患者，尤需注意避免暴露於下列何種物質中？
  1. 汞
  2. 砷
  3. 鉛
  4. 鉻
解答：D
<100-2-88> 接觸或暴露於重金屬之危害已陸續被發現與許多健康效應有關。下列各項組合中那一組之因果相關性最低？
  1. 錳－巴金森氏徵候群
  2. 汞－水俁病
  3. 砷－烏腳病
  4. 鉻－致畸胎
解答：D
<100-2-90> 戴奧辛在各項環境介質中的污染及其對國人健康造成的影響深受關注。下列何者為國內戴奧辛排放量最高之污染來源？
  1. 廢棄物焚化爐
  2. 金屬冶煉業
  3. 農田露天燃燒
  4. 有機氯農藥製造鉻
解答：B
<100-2-92> 工作場所中的物理性危害多來自於下列那一項？
  1. 產品材料
  2. 勞工長工時
  3. 特殊機具使用
  4. 工廠佈置
解答：C
<101-1-90> 下列那一項是對呼吸系統及眼部黏膜具刺激性之空氣污染物？
  1. CO2
  2. CO
  3. O3
  4. CH4
解答：C
<101-1-93> 目前所訂定的鉛中毒標準，對小孩族群之保護遠比對成人嚴格，主要來自下列那一系統或器官的毒性效應？
  1. 呼吸系統
  2. 周圍神經系統
  3. 造血系統
  4. 肝

<102-1-92> 下列那一種物質在飲水中含量過高時，對嬰兒會引發藍嬰症（ methemoglobinemia ）？
  1. 硝酸鹽
  2. 磷酸鹽
  3. 硫酸鹽 
  4. 氰酸鹽

<102-1-94> 一般評估環境品質的主要空氣污染物，包括下列那些物質？
  1. 臭氧、一氧化碳、懸浮微粒、二氧化硫、二氧化氮
  2. 多環芳香碳氫化合物、臭氧、一氧化碳、鉛、苯
  3. 甲醛、硫酸氫、石綿、苯、臭味物質
  4. 甲苯、二氧化硫、丙烯、一氧化碳、二氧化氮
解答：A
<102-2-8> 都會區光化學煙霧主要為空氣中碳氫化合物與下列何類物質反應生？
  1. 硫氧化物
  2. 氮氧化物
  3. 一氧化碳
  4. 氟氯碳化物

<102-2-94> 下列何者與石棉暴露的病史相關？
