    вАҐ (дЄА) дЇ§жШУжИРжЬђдєЛжДПзЊ©

    дЇ§жШУжИРжЬђпЉИ Transaction Cost пЉЙпЉМжМЗдЇ§жШУйБОз®ЛдЄ≠жЙАењЕй†Ии≤†жУФдєЛжИРжЬђгАВдї•и≤Ји≥£зВЇдЊЛпЉМи≤ЈжЦєењЕй†Ии≤†жУФдЇ§жШУжИРжЬђпЉМи≥£жЦєдЇ¶й†Ии≤†жУФдЇ§жШУжИРжЬђгАВ

    вАҐ (дЇМ) дЇ§жШУжИРжЬђдєЛеЕІеЃє

    2. йЫЩжЦєи≠∞еГєе∞Лж±ВзЬЯеѓ¶еГєж†ЉдєЛжИРжЬђпЉЪи≤Ји≥£йЫЩжЦєи®ОеГєйВДеГєпЉМжЬАеЊМйБФжИРдЄАиЗіеГєж†ЉгАВеЫ†ж≠§пЉМи≤Ји≥£йЫЩжЦєеЭЗжЬЙеНФеХЖи≠∞еГєдєЛжИРжЬђгАВ

    3. и®ВеЃЪе•СзіДдєЛжИРжЬђпЉЪи≤Ји≥£йЫЩжЦєз∞љи®Ве•СзіДпЉМењЕй†Ие∞Не•СзіДеЕІеЃєйАРжҐЭеѓ©и¶ЦпЉМдї•йБњеЕНе•СзіДеЕІеЃєдЄНеИ©жЦЉеЈ±пЉМиАМйА†жИРдЇ§жШУйҐ®йЪ™гАВеЫ†ж≠§пЉМи≤Ји≥£йЫЩжЦєеЭЗжЬЙз∞љи®Ве•СзіДдєЛжИРжЬђгАВ

    4. зЫ£и¶ЦиИЗеЯЈи°Ме•СзіДдєЛжИРжЬђпЉЪи≤Ји≥£йЫЩжЦєдЊЭе•СзіДи¶ПеЃЪи°МдљњжђКеИ©еПКе±•и°МзЊ©еЛЩгАВи≤ЈжЦєзЫ£и¶Це∞НжЦєпЉМжШѓеР¶е¶ВжЬЯиЊ¶зРЖйБОжИґзЩїи®ШпЉЫи≥£жЦєзЫ£и¶Це∞НжЦєпЉМжШѓеР¶е¶ВжЬЯдЇ§дїШжђЊй†ЕгАВеЫ†ж≠§пЉМи≤Ји≥£йЫЩжЦєеЭЗжЬЙзЫ£и¶Це∞НжЦєе±•и°МдєЛжИРжЬђгАВ

    5. дњЭи≠Ји≤°зФҐжђКжОТйЩ§дїЦдЇЇдЊµжђКдєЛжИРжЬђпЉЪе•СзіДе±•и°МзµРжЮЬпЉМи≤ЈжЦєеПЦеЊЧдЄНеЛХзФҐпЉМи≥£жЦєеПЦеЊЧеГєйЗСгАВе¶ВжЬЙдЄАжЦєйБХзіДпЉМеП¶дЄАжЦєи°МдљњиЂЛж±ВжђКпЉМи®іиЂЛж≥ХйЩҐеЉЈеИґеЯЈи°МгАВеЫ†ж≠§пЉМйЫЩжЦєеЭЗжЬЙдњЭи≠Ји≤°зФҐжђКжОТйЩ§дїЦдЇЇдЊµжђКдєЛжИРжЬђгАВ

    вАҐ (дЄЙ) дЄНеЛХзФҐдЇ§жШУжИРжЬђеБПйЂШ

    дЄНеЛХзФҐдЇ§жШУиИЗеЕґдїЦи≤°и≤®дЇ§жШУдЄАж®£пЉМеЭЗе≠ШеЬ®дЇ§жШУжИРжЬђгАВзДґпЉМдЄНеЛХзФҐдЇ§жШУжИРжЬђеБПйЂШпЉМеЕґзРЖзФ±е¶ВдЄЛпЉЪ

    1. дЄНеЛХзФҐеЕЈжЬЙзХ∞и≥™жАІпЉЪжѓПдЄАдїґдЄНеЛХзФҐйГљдЄНдЄАж®£пЉМеЫ†ж≠§и≤ЈжЦєи≥Љи≤ЈдЄНеЛХзФҐењЕй†Из≤ЊжМСзі∞йБЄпЉМдЄ¶иµізПЊе†ізЬЛе±ЛгАВ

    2. дЄНеЛХзФҐзД°йЫЖдЄ≠еЄВе†іпЉЪиВ°з•®жЬЙйЫЖдЄ≠еЄВе†іпЉМжХЕеЕґдЇ§жШУжИРжЬђдљОгАВдЄНеЛХзФҐзД°йЫЖдЄ≠еЄВе†іпЉМжХЕеЕґдЇ§жШУжИРжЬђйЂШгАВ

    3. дЄНеЛХзФҐи≥Зи®КдЄНжµБйАЪпЉЪдЄНеЛХзФҐдЇ§жШУи≥Зи®КиЉГеЕґдїЦи≤°и≤®е∞БйЦЙгАВдЄНеЛХзФҐдЇ§жШУи≥Зи®КдЄНжµБйАЪгАБдЄНж≠£зҐЇеПКдЄНе∞Нз®±пЉМйА†жИРдЄНеЛХзФҐдЇ§жШУжИРжЬђйЂШгАВ

    4. дЄНеЛХзФҐдЇ§жШУйЗСй°Не§ІпЉЪдЄНеЛХзФҐе±ђжЦЉйЂШеГєи≤°и≤®пЉМи≤Ји≥£дЇ§жШУй†ИзЙєеИ•иђєжЕОпЉМз®НжЬЙдЄНжЕОпЉМе∞ЗжРНе§±жЕШйЗНгАВеЫ†ж≠§пЉМдЄНеЛХзФҐдЇ§жШУдєЛи≠∞еГєжЩВйЦУйХЈпЉМй†РйШ≤жО™жЦље§ЪпЉМжХЕдЄНеЛХзФҐдЇ§жШУжИРжЬђйЂШгАВ

    5. дЄНеЛХзФҐйБОжИґжЙЛзЇМзєБзС£пЉЪдЄНеЛХзФҐдЇ§жШУењЕй†ИиЊ¶зРЖзІїиљЙзЩїи®ШгАВйБОжИґжЙЛзЇМзєБи§ЗгАБжЩВйЦУжЛЙйХЈгАБйҐ®йЪ™еҐЮйЂШпЉМйА†жИРдЄНеЛХзФҐдЇ§жШУжИРжЬђйЂШгАВ

    вАҐ (еЫЫ) дЇ§жШУжИРжЬђе∞НдЄНеЛХзФҐеЄВе†ідєЛељ±йЯњ

    е¶ВеЬЦжЙАз§ЇпЉМD0дї£и°®зД°дЇ§жШУжИРжЬђдєЛдЄНеЛХзФҐйЬАж±ВжЫ≤зЈЪпЉМS0дї£и°®зД°дЇ§жШУжИРжЬђдєЛдЄНеЛХзФҐдЊЫзµ¶жЫ≤зЈЪгАВзФ±D0иИЗS0дєЛдЇ§йїЮbпЉМж±ЇеЃЪдЄНеЛХзФҐдЇ§жШУжХЄйЗПQ0гАВзХґдЄНеЛХзФҐи≤Ји≥£е≠ШеЬ®дЇ§жШУжИРжЬђпЉМеЫ†и≤ЈжЦєжЬЙдЇ§жШУжИРжЬђиАМдљњD0еЈ¶дЄЛзІїиЗ≥D1пЉМеЫ†и≥£жЦєжЬЙдЇ§жШУжИРжЬђиАМдљњS0еЈ¶дЄКзІїиЗ≥S1гАВзФ±D1иИЗS1дєЛдЇ§йїЮeпЉМж±ЇеЃЪдЄНеЛХзФҐдЇ§жШУжХЄйЗПQ1гАВдЄНеЛХзФҐи≤Ји≥£еЫ†е≠ШеЬ®дЇ§жШУжИРжЬђпЉМдЄНеЛХзФҐдЇ§жШУйЗПзФ±Q0жЄЫе∞СиЗ≥Q1пЉМз§ЊжЬГз¶ПеИ©зФ±вЦ≥abcзЄЃе∞ПзВЇвЦ≥defгАВ
